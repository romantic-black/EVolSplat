## 关键概念速查

### 数据维度
- **输入图像**: `[N_views, H, W, 3]`
- **3D点云**: `[N_points, 3]`
- **渲染输出**: `[H, W, 3]`

### 核心组件
- **SparseCostRegNet**: 3D特征提取
- **Projector**: 2D特征采样
- **MLP解码器**: 特征融合和参数预测
- **Gaussian Splatting**: 可微渲染

### 训练配置
- **损失函数**: L1 + SSIM + Entropy
- **优化器**: 分组优化（6个参数组）
- **评估指标**: PSNR, SSIM, LPIPS
